OAuth::setTimestamp

(PECL OAuth >= 1.0.0)

OAuth::setTimestamp — 设置时间戳

说明

public mixed OAuth::setTimestamp ( string $timestamp )

为后续请求设置时间戳。

Warning

本函数还未编写文档，仅有参数列表。

参数

timestamp

时间戳。

返回值

返回 TRUE ，除非 timestamp 无效，则返回 FALSE 。

更新日志

版本 说明
1.0.0 以前失败时返回 NULL，而不是 FALSE。
