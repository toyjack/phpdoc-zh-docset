运行时配置

这些函数的行为受 php.ini 中的设置影响。

cURL 配置选项
名字 默认 可修改范围 更新日志
curl.cainfo NULL PHP_INI_SYSTEM 自 PHP 5.3.7. 起有效
有关 PHP_INI_* 样式的更多详情与定义，见 配置可被设定范围。

这是配置指令的简短说明。

curl.cainfo string

为 CURLOPT_CAINFO 选项设置一个默认值。 这必须是一个绝对路径。
