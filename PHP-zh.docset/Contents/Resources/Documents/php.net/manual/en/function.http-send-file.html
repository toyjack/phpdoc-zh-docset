http_send_file

(PECL pecl_http >= 0.1.0)

http_send_file — Send file

说明

bool http_send_file ( string $file )

Sends a file with support for (multiple) range requests.

This functions behaviour and further action is dependent on the following INI 设置s: http.send.not_found_404 和 http.log.not_found.

If the INI 设置 http.send.not_found_404 is enabled and the INI 设置 http.log.not_found points to a writable file, a log message is written when the file was not found.

参数

file

the file to send

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

范例

Example #1 A http_send_file() example

<?php
http_send_content_disposition("document.pdf", true);
http_send_content_type("application/pdf");
http_throttle(0.1, 2048);
http_send_file("../report.pdf");
?>
