pg_tty

(PHP 4, PHP 5)

pg_tty — 返回该连接的 tty 号

说明

string pg_tty ( resource $connection )

pg_tty() 返回指定 PostgreSQL connection 资源在服务器端输出调试信息的 tty 号。
