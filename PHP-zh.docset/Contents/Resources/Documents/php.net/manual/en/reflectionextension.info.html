ReflectionExtension::info

(PHP 5, PHP 7)

ReflectionExtension::info — 输出扩展信息

说明

public void ReflectionExtension::info ( void )

输出"phpinfo()"信息中的扩展信息。

参数

此函数没有参数。

返回值

扩展信息。

范例

Example #1 ReflectionExtension::info() example

<?php
$ext = new ReflectionExtension('mysqli');
$ext->info();
?>
