filter_list

(PHP 5 >= 5.2.0, PHP 7)

filter_list — 返回所支持的过滤器列表

说明

array filter_list ( void )

返回值

返回一个所支持的过滤器的名称的列表，如果没有这样子的过滤器的话则返回空数组。这个数组的索引不是过滤器id， 你可以通过 filter_id() 去根据名称获取它们。

范例

Example #1 一个 filter_list() 的例子

<?php
print_r(filter_list());
?>

以上例程的输出类似于：
