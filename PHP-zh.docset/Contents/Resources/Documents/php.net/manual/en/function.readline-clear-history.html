readline_clear_history

(PHP 4, PHP 5, PHP 7)

readline_clear_history — 清除历史

说明

bool readline_clear_history ( void )

这个函数清除整个的命令行历史.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
