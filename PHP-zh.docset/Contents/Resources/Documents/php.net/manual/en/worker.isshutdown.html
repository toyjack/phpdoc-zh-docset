Worker::isShutdown

(PECL pthreads >= 2.0.0)

Worker::isShutdown — 状态检测

说明

public boolean Worker::isShutdown ( void )

Worker 对象是否被关闭

参数

此函数没有参数。

返回值

布尔值，表示否是被关闭

范例

Example #1 检测 Worker 对象状态

<?php
$my = new Worker();
$my->start();
var_dump($my->isShutdown());
$my->shutdown();
var_dump($my->isShutdown());
?>

以上例程会输出：

bool(false)
bool(true)
