SQLite3Result::numColumns

(PHP 5 >= 5.3.0, PHP 7)

SQLite3Result::numColumns — Returns the number of columns in the result set

说明

public int SQLite3Result::numColumns ( void )

Returns the number of columns in the result set.

参数

此函数没有参数。

返回值

Returns the number of columns in the result set.
