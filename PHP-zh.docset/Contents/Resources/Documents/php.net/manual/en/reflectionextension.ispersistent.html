ReflectionExtension::isPersistent

(PHP >= 5.4.0)

ReflectionExtension::isPersistent — 返回扩展是否持久化载入

说明

public void ReflectionExtension::isPersistent ( void )

Warning

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

扩展在extension配置中被载入返回TRUE ，否则返回 FALSE。
