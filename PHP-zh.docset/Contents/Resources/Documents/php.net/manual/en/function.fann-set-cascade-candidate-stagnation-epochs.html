fann_set_cascade_candidate_stagnation_epochs — Sets the number of cascade candidate stagnation epochs

说明

bool fann_set_cascade_candidate_stagnation_epochs ( resource $ann , int $cascade_candidate_stagnation_epochs )

Sets the number of cascade candidate stagnation epochs.

参数

ann
