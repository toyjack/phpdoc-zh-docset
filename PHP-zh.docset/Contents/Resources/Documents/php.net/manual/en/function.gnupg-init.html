gnupg_init

(PECL gnupg >= 0.4)

gnupg_init — Initialize a connection

说明

resource gnupg_init ( void )

参数

此函数没有参数。

返回值

A GnuPG resource connection used by other GnuPG functions.

范例

Example #1 Procedural gnupg_init() example

<?php
$res = gnupg_init();
?>

Example #2 OO gnupg initializer example

<?php
$gpg = new gnupg();
?>
