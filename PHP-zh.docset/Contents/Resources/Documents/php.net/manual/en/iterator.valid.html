Iterator::valid

(PHP 5 >= 5.0.0, PHP 7)

Iterator::valid — 检查当前位置是否有效

说明

abstract public boolean Iterator::valid ( void )

此方法在 Iterator::rewind() 和 Iterator::next() 方法之后被调用以此用来检查当前位置是否有效。

参数

此函数没有参数。

返回值

返回将被转换为布尔型。成功时返回 TRUE， 或者在失败时返回 FALSE。
