streamWrapper::__destruct

(PHP 4 >= 4.3.2, PHP 5)

streamWrapper::__destruct — Destructs an existing stream wrapper

说明

streamWrapper::__destruct ( void )

Called when closing the stream wrapper, right before streamWrapper::stream_flush().

参数

此函数没有参数。
