Event::addTimer

(PECL event >= 1.2.6-beta)

Event::addTimer — Makes timer event pending

说明

public bool Event::addTimer ([ double $timeout ] )

Event::addTimer() is an alias of Event::add()

返回值

范例

Example #1 Event::addTimer() example

<?php
$base = new EventBase();
$n = 2;
$e = Event::timer($base, function($n) use (&$e) {
    echo "$n seconds elapsed\n";
    $e->delTimer();
}, $n);
$e->addTimer($n);
$base->loop();
?>

以上例程的输出类似于：

2 seconds elapsed

参见

  • Event::add() - Makes event pending
  • Event::del() - Makes event non-pending
  • Event::delTimer() - Makes timer event non-pending
