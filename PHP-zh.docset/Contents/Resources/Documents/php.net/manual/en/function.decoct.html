decoct

(PHP 4, PHP 5)

decoct — 十进制转换为八进制

说明

string decoct ( int $number )

返回一字符串，包含有给定 number 参数的八进制表示。所能转换的最大数值为十进制的 4294967295，其结果为 "37777777777"。

参数

number

待转换的十进制值

返回值

number 参数八进制表示的字符串。

范例

Example #1 decoct() 范例

<?php
echo decoct(15) . "\n";
echo decoct(264);
?>

以上例程会输出：

17
410
