Thread::start

(PECL pthreads >= 2.0.0)

Thread::start — 执行

说明

public boolean Thread::start ([ integer $options ] )

在独立线程中执行 run 方法

参数

options

可选参数，用来控制线程继承。默认值为 PTHREADS_INHERIT_ALL

返回值

布尔值，表示操作成功与否

范例

Example #1 开始线程

<?php
class My extends Thread {
    public function run() {
        /** ... **/
    }
}
$my = new My();
var_dump($my->start());
?>

以上例程会输出：

bool(true)
