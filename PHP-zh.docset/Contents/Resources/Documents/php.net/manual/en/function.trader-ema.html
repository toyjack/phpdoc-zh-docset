(PECL trader >= 0.2.0)

trader_ema — Exponential Moving Average

说明

array trader_ema ( array $real [, integer $timePeriod ] )

参数

real

浮点数数组。

timePeriod

Number of period. Valid range from 2 to 100000.

返回值

Returns an array with calculated data or false on failure.
