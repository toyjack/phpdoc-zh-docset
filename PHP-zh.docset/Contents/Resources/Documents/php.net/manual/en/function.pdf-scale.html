(PHP 4, PECL pdflib >= 1.0.0)

PDF_scale — Scale coordinate system

说明

bool PDF_scale ( resource $p , float $sx , float $sy )

Scales the coordinate system. 成功时返回 TRUE， 或者在失败时返回 FALSE。
