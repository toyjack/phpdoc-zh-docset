Cairo::versionString

cairo_version_string

(PECL cairo >= 0.1.0)

Cairo::versionString -- cairo_version_string — Retrieves cairo version as string

说明

面向对象风格 (method):

public static string Cairo::versionString ( void )

过程化风格:

string cairo_version_string ( void )

Retrieves the current cairo library version as a string.

参数

此函数没有参数。

返回值

Current Cairo library version string

范例

Example #1 面向对象风格

<?php

var_dump(Cairo::versionString());

?>

以上例程的输出类似于：

string(5) "1.8.8"

Example #2 过程化风格

<?php

var_dump(cairo_version_string());

?>

以上例程的输出类似于：

string(5) "1.8.8"
