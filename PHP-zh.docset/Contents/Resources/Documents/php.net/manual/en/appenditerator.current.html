AppendIterator::current

(PHP 5 >= 5.1.0)

AppendIterator::current — Gets the current value

说明

public mixed AppendIterator::current ( void )

Gets the current value.

参数

此函数没有参数。

返回值

The current value if it is valid or NULL otherwise.
