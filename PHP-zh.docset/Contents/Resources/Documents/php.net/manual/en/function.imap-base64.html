imap_base64

(PHP 4, PHP 5)

imap_base64 — Decode BASE64 encoded text

说明

string imap_base64 ( string $text )

Decodes the given BASE-64 encoded text.

参数

text

The encoded text

返回值

Returns the decoded message as a string.
