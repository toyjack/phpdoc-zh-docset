ibase_free_result

(PHP 5)

ibase_free_result — Free a result set

说明

bool ibase_free_result ( resource $result_identifier )

Frees a result set.

参数

result_identifier

A result set created by ibase_query() or ibase_execute().

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
