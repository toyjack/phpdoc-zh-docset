stream_get_transports

(PHP 5, PHP 7)

stream_get_transports — 获取已注册的套接字传输协议列表

说明

array stream_get_transports ( void )

返回一个包含当前运行系统中所有套接字传输协议名称的索引数组。

返回值

返回一个套接字传输协议名称的索引数组。

范例

Example #1 使用 stream_get_transports()

<?php
$xportlist = stream_get_transports();
print_r($xportlist);
?>

以上例程的输出类似于：

Array (
  [0] => tcp
  [1] => udp
  [2] => unix
  [3] => udg
)
