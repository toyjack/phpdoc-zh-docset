imagepsextendfont

(PHP 4, PHP 5)

imagepsextendfont — 扩充或精简字体

说明

bool imagepsextendfont ( resource $font_index , float $extend )

扩充或精简某字体（font_index）。如果 extend 参数的值小于一则是精简该字体。

参数

font_index

由 imagepsloadfont() 返回的字体资源。

extend

扩展的值，必须大于 0。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。

范例

Example #1 imagepsextendfont() 例子

<?php
// Load a .pfb font file
$font = imagepsloadfont('./px3l.pfb');

// Extend the font by 2.5
imagepsextendfont($font, 2.5);

// Do any operations with the font here

// Free the font from memory
imagepsfreefont($font);
?>

注释

Note: 此函数仅在 PHP 编译时指定了 --with-t1lib[=DIR] 时可用。
