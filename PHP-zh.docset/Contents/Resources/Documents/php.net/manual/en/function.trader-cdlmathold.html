trader_cdlmathold

(PECL trader >= 0.2.0)

trader_cdlmathold — Mat Hold

说明

array trader_cdlmathold ( array $open , array $high , array $low , array $close [, float $penetration ] )

参数

open

开盘价，浮点数数组。

high

高价，浮点数数组。

low

低价，浮点数数组。

close

收盘价，浮点数数组。

penetration

一个烛台渗入到另一个烛台的百分比。

返回值

Returns an array with calculated data or false on failure.
