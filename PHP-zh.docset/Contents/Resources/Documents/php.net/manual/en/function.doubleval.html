doubleval

(PHP 4, PHP 5, PHP 7)

doubleval — floatval() 的别名

描述

此函数是 floatval() 的别名。

Note:

此别名是函数改名之后的遗留问题。在 PHP 旧的版本中由于还没有 floatval() 函数，所以你可能需要用到这个 floatval() 的别名函数。
