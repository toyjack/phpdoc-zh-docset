Ubuntu 12.04 and up no longer has php5-sqlite3 included.

So this will no loger work:
<?php
$db = sqlite3_open(":memory:");
?>

Use this:
<?php
$db = new SQLite3('mysqlitedb.db');

$results = $db->query('SELECT bar FROM foo');
while ($row = $results->fetchArray()) {
    var_dump($row);
}
?>
