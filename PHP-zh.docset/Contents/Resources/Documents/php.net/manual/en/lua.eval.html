Lua::eval

(PECL lua >=0.9.0)

Lua::eval — 将字符串当做Lua代码执行

说明

public mixed Lua::eval ( string $statements )

Warning

本函数还未编写文档，仅有参数列表。

参数

statements

返回值

返回运行结果，参数错误返回NULL ，其它错误返回FALSE。

范例

Example #1 Lua::eval()示例

<?php
$lua = new Lua();
$lua->eval(<<<CODE
    print(2);
CODE
);
?>

以上例程会输出：

2
