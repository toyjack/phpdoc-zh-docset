    Generator::getReturn

    (PHP 7)

    Generator::getReturn â€” Get the return value of a generator

    Description

    public mixed Generator::getReturn ( void )

    Parameters

    This function has no parameters.

    Return Values

    Returns the generator's return value once it has finished executing.

    Examples

    Example #1 Generator::getReturn() example

    <?php

    $gen     = (function() {
            yield 1;
            yield 2;

        return     3;
    })();

    foreach (    $gen as $val) {
        echo     $val, PHP_EOL;
    }

    echo     $gen->getReturn(), PHP_EOL;

    The above example will output:

    1
2
3

    Be careful that `->getReturn()` will throw an exception if no return is present in the generator:

    <?php
    function g() {
        yield     'A';
    }
    try {
            g()->getReturn();
    } catch (    Exception $ex) {
            var_dump($ex->getMessage());
            // Cannot get return value of a generator that hasn't returned
    }
    ?>
