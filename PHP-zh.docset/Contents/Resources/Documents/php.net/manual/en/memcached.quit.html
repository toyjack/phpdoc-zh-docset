Memcached::quit

(PECL memcached >= 2.0.0)

Memcached::quit — 关闭所有打开的链接。

说明

public bool Memcached::quit ( void )

Memcached::quit() 关闭所有memcache服务器的链接。

参数

此函数没有参数。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
