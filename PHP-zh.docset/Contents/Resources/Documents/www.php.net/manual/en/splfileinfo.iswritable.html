SplFileInfo::isWritable

(PHP 5 >= 5.1.2, PHP 7)

SplFileInfo::isWritable — Tells if the entry is writable

说明

public SplFileInfo::isWritable ( ) : bool

Checks if the current entry is writable.

参数

此函数没有参数。

返回值

Returns true if writable, false otherwise;
