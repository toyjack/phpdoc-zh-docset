mcrypt_module_close

(PHP 4 >= 4.0.2, PHP 5, PHP 7 < 7.2.0, PECL mcrypt >= 1.0.0)

mcrypt_module_close — 关闭加密模块

Warning

本函数已自 PHP 7.1.0 起废弃。强烈建议不要使用本函数。

说明

mcrypt_module_close(resource $td): bool

关闭加密模块。

参数

td

加密描述符。

返回值

成功时返回 true， 或者在失败时返回 false。
