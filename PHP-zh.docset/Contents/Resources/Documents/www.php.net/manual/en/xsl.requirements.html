需求

此扩展需要 libxml PHP 扩展。这表示需要使用 --enable-libxml ，尽管这将隐式完成因为 libxml 是缺省开启的。

This extension uses libxslt which can be found at » http://xmlsoft.org/XSLT/. libxslt version 1.1.0 or greater is required.
