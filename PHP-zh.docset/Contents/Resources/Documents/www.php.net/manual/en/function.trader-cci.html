trader_cci

(PECL trader >= 0.2.0)

trader_cci — Commodity Channel Index

说明

trader_cci(
    array $high,
    array $low,
    array $close,
    int $timePeriod = ?
): array

参数

high

高价，浮点数数组。

low

低价，浮点数数组。

close

收盘价，浮点数数组。

timePeriod

Number of period. Valid range from 2 to 100000.

返回值

Returns an array with calculated data or false on failure.
