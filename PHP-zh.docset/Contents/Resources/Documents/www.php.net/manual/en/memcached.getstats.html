Memcached::getStats

(PECL memcached >= 0.1.0)

Memcached::getStats — 获取服务器池的统计信息

说明

public Memcached::getStats ( void ) : array

Memcached::getStats()返回一个包含所有可用memcache服务器状态的数组. 返回的统计信息的详细描述参见» memcache protocol。 （译注：经实验，服务器池中有不可用服务器时，返回false）

参数

此函数没有参数。

返回值

服务器统计信息数组， 每个服务器一项。

范例

Example #1 Memcached::getStats() 示例

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

print_r($m->getStats());
?>
