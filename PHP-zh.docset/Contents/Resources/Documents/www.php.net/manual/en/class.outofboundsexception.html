The OutOfBoundsException class

(PHP 5 >= 5.1.0, PHP 7)

简介

Exception thrown if a value is not a valid key. This represents errors that cannot be detected at compile time.

类摘要

OutOfBoundsException extends RuntimeException {
/* 继承的属性 */
