Error::getCode

(PHP 7, PHP 8)

Error::getCode — 获取错误代码

说明

final public Error::getCode ( ) : mixed

返回错误代码。

参数

此函数没有参数。

返回值

返回 integer 的错误代码

范例

Example #1 Error::getCode() 例子

<?php
try {
    throw new Error("Some error message", 30);
} catch(Error $e) {
    echo "The Error code is: " . $e->getCode();
}
?>

以上例程的输出类似于：

The Error code is: 30
