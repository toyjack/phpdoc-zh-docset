trader_rsi

(PECL trader >= 0.2.0)

trader_rsi — Relative Strength Index

说明

trader_rsi ( array $real [, int $timePeriod ] ) : array

参数

real

浮点数数组。

timePeriod

Number of period. Valid range from 2 to 100000.

返回值

Returns an array with calculated data or false on failure.
