HaruDoc::loadJPEG

(PECL haru >= 0.0.1)

HaruDoc::loadJPEG — Load a JPEG image

说明

HaruDoc::loadJPEG ( string $filename ) : object

Loads the specified JPEG image.

参数

filename

A valid JPEG image file.

返回值

Returns a new HaruImage instance.

错误／异常

发生错误时抛出 HaruException 异常。
