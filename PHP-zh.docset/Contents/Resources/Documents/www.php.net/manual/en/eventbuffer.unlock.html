EventBuffer::unlock

(PECL event >= 1.2.6-beta)

EventBuffer::unlock — Releases lock acquired by EventBuffer::lock

说明

public EventBuffer::unlock ( void ) : bool

Releases lock acquired by EventBuffer::lock() .

参数

此函数没有参数。

返回值

Returns TRUE on success. Otherwise FALSE.
