Directory::rewind

(PHP 4, PHP 5, PHP 7)

Directory::rewind — 倒回目录句柄

说明

public Directory::rewind ([ resource $dir_handle ] ) : void

与 rewinddir() 函数功能一样, 只是 dir_handle 默认为 $this 变量。
