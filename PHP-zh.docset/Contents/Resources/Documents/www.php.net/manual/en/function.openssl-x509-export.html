openssl_x509_export

(PHP 4 >= 4.2.0, PHP 5, PHP 7)

openssl_x509_export — 以字符串格式导出证书

说明

openssl_x509_export ( mixed $x509 , string &$output [, bool $notext = TRUE ] ) : bool

openssl_x509_export() 将 x509 以PEM编码的格式导出到名为 output 的字符串类型的变量中。

参数

x509

参见密钥／证书参数以获取有效值列表。

output

成功，将会存储 PEM.

notext

可选参数 notext 影响输出的冗余度。如果设为 FALSE，输出内容将包含附加的人类可读信息。notext 的缺省值为 TRUE。

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
