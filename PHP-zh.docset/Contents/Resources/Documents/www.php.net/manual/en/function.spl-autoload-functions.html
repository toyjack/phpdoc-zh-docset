spl_autoload_functions

(PHP 5 >= 5.1.0, PHP 7)

spl_autoload_functions — 返回所有已注册的 __autoload() 函数

说明

spl_autoload_functions ( ) : array

获取所有已注册的 __autoload() 函数。

参数

此函数没有参数。

返回值

包含所有已注册的 __autoload 函数的数组（array）。如果自动装载函数队列未激活，则返回 false。如果没有已注册的函数，则返回一个空数组。
