apcu_key_info — Get detailed information about the cache key

说明

apcu_key_info ( string $key ) : array

Get detailed information about the cache key

参数

key

Get detailed information about the cache key

返回值

null or array info

范例

Example #1 A apcu_key_info() example

<?php
apcu_add('a','b');
var_dump(apcu_key_info('a'));
?>

以上例程会输出：

array(7) {
  ["hits"]=>
  int(0)
  ["access_time"]=>
  int(1606701783)
  ["mtime"]=>
  int(1606701783)
  ["creation_time"]=>
  int(1606701783)
  ["deletion_time"]=>
  int(0)
  ["ttl"]=>
  int(0)
  ["refs"]=>
  int(0)
}
