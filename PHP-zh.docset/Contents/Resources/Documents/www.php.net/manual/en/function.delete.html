delete — 参见 unlink() 或 unset()

说明

在 PHP 语言里，没有 delete 关键词或函数。在这里，你若要删除文件，可以使用 unlink()。在本地作用域删除变量可使用 unset()。

参见

  • unlink() - 删除文件
  • unset() - 释放给定的变量
