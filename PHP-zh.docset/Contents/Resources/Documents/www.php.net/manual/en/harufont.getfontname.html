HaruFont::getFontName

(PECL haru >= 0.0.1)

HaruFont::getFontName — Get the name of the font

说明

HaruFont::getFontName ( void ) : string

Get the name of the font.

参数

此函数没有参数。

返回值

Returns the name of the font.

错误／异常

发生错误时抛出 HaruException 异常。
