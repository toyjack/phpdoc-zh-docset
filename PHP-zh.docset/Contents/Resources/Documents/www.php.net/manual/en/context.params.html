Context 参数

Context 参数 — Context 参数列表

说明

这些参数（parameters）可以设置为由函数 stream_context_set_params() 返回的 context。

参数

notification callable

当一个流（stream）上发生事件时，callable 将被调用。

查看 stream_notification_callback 以获得更多信息。
