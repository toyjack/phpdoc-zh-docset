PDF_fill_stroke

(PHP 4, PECL pdflib >= 1.0.0)

PDF_fill_stroke — Fill and stroke path

说明

PDF_fill_stroke ( resource $p ) : bool

Fills and strokes the current path with the current fill and stroke color. 成功时返回 TRUE， 或者在失败时返回 FALSE。
