gmp_perfect_power

(PHP 7 >= 7.3.0, PHP 8)

gmp_perfect_power — Perfect power check

说明

gmp_perfect_power ( GMP|int|string $num ) : bool

Checks whether num is a perfect power.

参数

num

GMP 对象或 int ，或数字string。

返回值

Returns true if num is a perfect power, false otherwise.
