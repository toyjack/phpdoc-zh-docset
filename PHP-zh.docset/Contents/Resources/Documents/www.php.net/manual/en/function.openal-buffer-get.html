openal_buffer_get

(PECL openal >= 0.1.0)

openal_buffer_get — Retrieve an OpenAL buffer property

说明

openal_buffer_get(resource $buffer, int $property): int|false

参数

buffer

An Open AL(Buffer) resource (previously created by openal_buffer_create()).

property

Specific property, one of: AL_FREQUENCY, AL_BITS, AL_CHANNELS 和 AL_SIZE.

返回值

Returns an integer value appropriate to the property requested 或者在失败时返回 false.

参见

  • openal_buffer_create() - Generate OpenAL buffer
