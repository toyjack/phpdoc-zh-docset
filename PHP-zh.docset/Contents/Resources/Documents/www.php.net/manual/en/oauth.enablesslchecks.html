OAuth::enableSSLChecks

(PECL OAuth >= 0.99.5)

OAuth::enableSSLChecks — 启用 SSL 检查

说明

public OAuth::enableSSLChecks(): bool

启用通常的l SSL 对等证书和主机检查（默认为启用）。或者，可以设置 sslChecks 成员为一个非 false 值来启用 SSL 检查。

参数

此函数没有参数。

返回值

true
