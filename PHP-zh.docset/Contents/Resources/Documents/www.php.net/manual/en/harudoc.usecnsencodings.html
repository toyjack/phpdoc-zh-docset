HaruDoc::useCNSEncodings

(PECL haru >= 0.0.1)

HaruDoc::useCNSEncodings — Enable Chinese simplified encodings

说明

HaruDoc::useCNSEncodings ( void ) : bool

Enables Chinese simplified encodings.

参数

此函数没有参数。

返回值

Returns TRUE on success.

错误／异常

发生错误时抛出 HaruException 异常。
