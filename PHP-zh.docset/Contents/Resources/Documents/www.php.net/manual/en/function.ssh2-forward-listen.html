ssh2_forward_listen

(PECL ssh2 >= 0.9.0)

ssh2_forward_listen — Bind a port on the remote server and listen for connections

说明

ssh2_forward_listen(
    resource $session,
    int $port,
    string $host = ?,
    int $max_connections = 16
): resource|false

Binds a port on the remote server and listen for connections.

Warning

本函数还未编写文档，仅有参数列表。

参数

session

An SSH Session resource, obtained from a call to ssh2_connect().

port

The port of the remote server.

host

max_connections

返回值

Returns an SSH2 Listener, 或者在失败时返回 false.
