IntlCalendar::before

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::before — Whether this object?s time is before that of the passed object

说明

面向对象风格

public IntlCalendar::before(IntlCalendar $other): bool

过程化风格

intlcal_before(IntlCalendar $calendar, IntlCalendar $other): bool

Returns whether this object?s time precedes the argument?s time.

参数

calendar

IntlCalendar 实例。

other

The calendar whose time will be checked against the primary object?s time.

返回值

Returns true if this object?s current time is before that of the calendar argument?s time. Returns false otherwise.

On failure false is also returned. To detect error conditions use intl_get_error_code(), or set up Intl to throw exceptions.
