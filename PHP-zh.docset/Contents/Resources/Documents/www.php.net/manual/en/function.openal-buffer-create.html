openal_buffer_create

(PECL openal >= 0.1.0)

openal_buffer_create — Generate OpenAL buffer

说明

openal_buffer_create(): resource

参数

此函数没有参数。

返回值

Returns an Open AL(Buffer) resource on success or false on failure.
