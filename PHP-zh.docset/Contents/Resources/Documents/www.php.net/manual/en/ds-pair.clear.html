Ds\Pair::clear — Removes all values

说明

public Ds\Pair::clear ( void ) : void

Removes all values from the pair.

参数

此函数没有参数。

返回值

没有返回值。

范例

Example #1 Ds\Pair::clear() example

<?php
$pair = new \Ds\Pair("a", 1);
print_r($pair);

$pair->clear();
print_r($pair);
?>

以上例程的输出类似于：

Ds\Pair Object
(
    [0] => 1
    [1] => 2
    [2] => 3
)
Ds\Pair Object
(
)
