Mongo::connectUtil

(PECL mongo >=0.9.0)

Mongo::connectUtil — Connects with a database server

说明

protected Mongo::connectUtil ( void ) : bool
Warning

This is an internal function that you should never call yourself.

参数

此函数没有参数。

返回值

If the connection was successful.

错误／异常

Throws MongoConnectionException if it fails to connect to the databases.
