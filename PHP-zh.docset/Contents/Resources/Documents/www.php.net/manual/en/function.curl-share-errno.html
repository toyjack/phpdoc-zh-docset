curl_share_errno

(PHP 7 >= 7.1.0, PHP 8)

curl_share_errno — 返回共享 curl 句柄的最后一次错误号

说明

curl_share_errno ( resource $sh ) : int

返回一个整数，表示共享 curl 句柄的最后一次错误号

参数

sh

由 curl_share_init() 函数创建的共享 cURL 句柄。

返回值

返回一个整数，表示共享 curl 句柄的最后一次错误号， 或者在失败时返回 false。
