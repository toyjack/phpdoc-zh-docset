ifxus_close_slob

(PHP 4, PHP 5 < 5.2.1)

ifxus_close_slob — Deletes the slob object

说明

ifxus_close_slob ( int $bid ) : bool

Deletes the slob object on the given slob object-id bid.

参数

bid

An existing slob id.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
