Thread::isStarted

(PECL pthreads >= 2.0.0)

Thread::isStarted — 状态检测

说明

public Thread::isStarted ( ) : bool

线程是否开始执行

参数

此函数没有参数。

返回值

布尔值，表示线程是否开始执行

范例

Example #1 监测线程是否开始执行

<?php
$worker = new Worker();
$worker->start();
var_dump($worker->isStarted());
?>

以上例程会输出：

bool(true)
