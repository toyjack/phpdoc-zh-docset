MongoCollection::__construct

(PECL mongo >=0.9.0)

MongoCollection::__construct — 创建一个新的集合

说明

public MongoCollection::__construct ( MongoDB $db , string $name )

参数

MongoDB db

所属的父数据库。

name

集合的名称。

返回值

返回一个新的集合对象。

错误／异常

如果集合名称无效，将会抛出一个默认的异常。
