说明

public Gmagick::despeckleimage ( void ) : Gmagick

Reduces the speckle noise in an image while preserving the edges of the original image.

参数

此函数没有参数。

返回值

The despeckled Gmagick object on success.

错误／异常

错误时抛出 GmagickException。
