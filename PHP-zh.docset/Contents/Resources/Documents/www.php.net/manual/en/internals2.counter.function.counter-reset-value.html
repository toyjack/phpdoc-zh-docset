counter_reset_value

counter_reset_value — 重置计数器资源的当前值。

说明

counter_reset_value ( resource $counter ) : void

counter_reset_value() 将计数器资源的当前值重置为其初始值。

参数

counter
要操作的计数器资源。

参见

  • counter_get_value() - 获取计数器资源的当前值。
  • counter_bump_value() - 更新计数器资源的当前值。
