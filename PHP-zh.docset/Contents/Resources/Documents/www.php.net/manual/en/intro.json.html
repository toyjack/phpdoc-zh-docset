简介

本扩展实现了 » JavaScript对象符号（JSON） 数据转换格式。 PHP 5 中的解码分析器基于 Douglas Crockford 的 JSON_checker。 PHP 7 中是改进的全新解析器，专门为 PHP 订制，软件许可证为 PHP license。

Note:

PHP 实现了 JSON 的一个超集，参考 » RFC 7159.
