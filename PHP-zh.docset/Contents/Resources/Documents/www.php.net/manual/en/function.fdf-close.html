fdf_close

(PHP 4, PHP 5 < 5.3.0, PECL fdf SVN)

fdf_close — Close an FDF document

说明

fdf_close ( resource $fdf_document ) : void

Closes the FDF document.

参数

fdf_document

The FDF document handle, returned by fdf_create(), fdf_open() or fdf_open_string().

返回值

没有返回值。
