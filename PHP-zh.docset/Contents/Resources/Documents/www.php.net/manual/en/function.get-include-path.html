get_include_path

(PHP 4 >= 4.3.0, PHP 5, PHP 7)

get_include_path — 获取当前的 include_path 配置选项

说明

get_include_path ( void ) : string

获取当前 include_path 配置选项的值。

返回值

返回字符串的路径。

范例

Example #1 get_include_path() 例子

<?php
echo get_include_path();

// 或使用 ini_get()
echo ini_get('include_path');
?>
