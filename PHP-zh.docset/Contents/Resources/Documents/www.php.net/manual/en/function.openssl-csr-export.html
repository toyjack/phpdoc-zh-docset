openssl_csr_export

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

openssl_csr_export — 将CSR作为字符串导出

说明

openssl_csr_export(mixed $csr, string &$out, bool $notext = true): bool

openssl_csr_export() 获取证书签名请求(csr)并通过引用保存其 PEM 格式的字符串(out)。

参数

csr

See CSR parameters for a list of valid values.

out

在成功时，该字符串将包含PEM编码的CSR.

notext

可选参数 notext 影响输出的冗余度。如果设为 false，输出内容将包含附加的人类可读信息。notext 的缺省值为 true。

返回值

成功时返回 true， 或者在失败时返回 false。

范例

Example #1 openssl_csr_export() 范例

<?php
$subject = array(
    "commonName" => "example.com",
);
$private_key = openssl_pkey_new(array(
    "private_key_bits" => 2048,
    "private_key_type" => OPENSSL_KEYTYPE_RSA,
));
$configargs = array(
    'digest_alg' => 'sha256WithRSAEncryption'
);
$csr = openssl_csr_new($subject, $private_key, $configargs);
openssl_csr_export($csr, $csr_string);
echo $csr_string;
?>
