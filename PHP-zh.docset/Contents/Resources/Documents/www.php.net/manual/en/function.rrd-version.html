rrd_version

(PECL rrd >= 1.0.0)

rrd_version — Gets information about underlying rrdtool library

说明

rrd_version ( void ) : string

Returns information about underlying rrdtool library.

参数

此函数没有参数。

返回值

String with rrdtool version number e.g. "1.4.3".
