mysqli::more_results

mysqli_more_results

(PHP 5, PHP 7, PHP 8)

mysqli::more_results -- mysqli_more_results — 检查批量查询中是否还有查询结果

说明

面向对象风格

mysqli::more_results(): bool

过程化风格

mysqli_more_results(mysqli $link): bool

检查上一次调用 mysqli_multi_query() 函数之后， 是否还有更多的查询结果集。

参数

mysql

仅以过程化样式：由mysqli_connect() 或 mysqli_init() 返回的 mysqli 对象。

返回值

如果上一次调用 mysqli_multi_query() 函数之后， 还有更多的结果集可以读取，返回 true，否则返回 false。

范例

参见 mysqli_multi_query()。

参见

  • mysqli_next_result() - 为读取 multi_query 执行之后的下一个结果集做准备
  • mysqli_store_result() - 转移上一次查询返回的结果集
  • mysqli_use_result() - Initiate a result set retrieval
