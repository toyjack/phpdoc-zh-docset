counter_get

counter_get — 获取简单计数器的当前值。

说明

counter_get ( void ) : integer

counter_get() 返回使用简单接口的计数器的当前值。

返回值

counter_get() 返回一个整数。

参见

  • counter_bump() - 修改简单计数器的当前值。
  • counter_reset() - 重置简单计数器的当前值。
