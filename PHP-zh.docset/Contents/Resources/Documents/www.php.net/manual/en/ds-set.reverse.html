Ds\Set::reverse

(PECL ds >= 1.0.0)

Ds\Set::reverse — Reverses the set in-place

说明

public Ds\Set::reverse(): void

Reverses the set in-place.

参数

此函数没有参数。

返回值

没有返回值。

范例

Example #1 Ds\Set::reverse() example

<?php
$set = new \Ds\Set(["a", "b", "c"]);
$set->reverse();

print_r($set);
?>

以上例程的输出类似于：

Ds\Set Object
(
    [0] => c
    [1] => b
    [2] => a
)
