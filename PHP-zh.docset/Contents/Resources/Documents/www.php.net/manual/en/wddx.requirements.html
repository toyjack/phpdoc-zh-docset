需求

此扩展需要 libxml PHP 扩展。这表示需要使用 --enable-libxml ，尽管这将隐式完成因为 libxml 是缺省开启的。

In order to use WDDX, you will need to install the expat library (which comes with Apache 1.3.7 or higher).
